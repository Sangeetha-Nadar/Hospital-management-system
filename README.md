**Patient Module**
Book and cancel appointments
Select doctors based on treatment specialization
Make secure online payments
Receive real-time email confirmation for appointment booking and payments
Chat with specialized doctors(can chat only with doctors who is specialist in the treatment needed for patient)
Provide ratings and feedback
View hospital information (About, Help, Policies, Description etc.)
View all kind of treatment available and their details.


 **Doctor Module**
View patients based on specialization
Receive messages only from patients related to their treatment domain
Reply to patient queries
Add hospital news (visible in UI scrolling section)
View admin notices


 **Receptionist Module**
Add, delete, and discharge patients
Manage patient records and payment details
Generate bills
View admin notices


 **Admin Module**
Add and manage hospital staff (Doctors, Nurses, Receptionists)
Send internal notices to staff
Manage overall system operations


 **Billing & Payment System**
Digital billing management
Payment records stored in database
Automated real-time email confirmation upon successful payment and appointment booking


 **System Highlights**
Role-based authentication
Secure database integration
Specialization-based doctor-patient communication
Real-time notification system
Structured workflow automation


**Technologies Used**
Frontend: HTML, CSS, JavaScript
Backend: C#
Database: MySQL
